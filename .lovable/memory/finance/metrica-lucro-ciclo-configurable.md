# Memory: finance/metrica-lucro-ciclo-configurable
Updated: now

O sistema permite configurar a 'Métrica de Lucro do Ciclo' por projeto via campo `metrica_lucro_ciclo` na tabela `projetos` (valores: 'operacional' | 'realizado', default: 'operacional'). (1) Operacional: Lucro Líquido = Apostas Liquidadas + Cashback + Giros − Perdas Confirmadas. (2) Realizado: Fluxo Líquido Ajustado = Saques − (Depósitos + Créditos Extras). Os Créditos Extras incluem: cashback, giros grátis, bônus creditados, ajustes de saldo e resultados cambiais. Esta configuração é definida no cadastro/edição do projeto e herdada por todos os ciclos. O serviço `calcularMetricasPeriodo.ts` retorna ambas as métricas (lucroLiquido e lucroRealizado) e os consumidores selecionam a apropriada com base na configuração do projeto.
